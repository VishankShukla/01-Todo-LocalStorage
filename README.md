01 - Todo LocalStorage

A simple and beginner-friendly Todo List web application built using HTML, CSS, and JavaScript, which uses the browser’s LocalStorage to store data persistently.

This project demonstrates basic CRUD operations (Create, Read, Update, Delete) without using any backend or database. All tasks are saved locally in the user's browser, so they remain available even after refreshing or reopening the page.

🚀 Features

Add new tasks

View all tasks

Delete tasks

Persistent storage using LocalStorage

Clean and simple user interface
